Overview
SmartFin RiskGuard is a machine learning–based financial application that helps users:

Track daily expenses

Analyze weekly & monthly spending

Predict financial risk

Detect suspicious QR transactions before payment

This system combines personal finance management + fraud detection into a single platform.

🚀 Features
🔐 User Authentication
Register, Login, Reset Password

Secure session handling

💰 Expense Tracking
Add daily expenses (Food, Shopping, etc.)

Stores user spending history

📊 Dashboard Analytics
7-day expense report

30-day monthly report

Category-wise spending charts

⚠ Financial Alerts
High spending warning

EMI impact detection

Low savings alerts

📈 Financial Risk Prediction
Uses Random Forest model

Predicts: Low / Medium / High risk

🔍 QR Fraud Detection
Scan QR (image upload)

Decode payment data

Validate UPI format

Detect fraud using Isolation Forest

Shows fraud risk score

📜 Transaction History
Stores QR payment records

Displays user history

🧠 Machine Learning Models
Model	Purpose
Random Forest	Financial Risk Prediction
Isolation Forest	Fraud / Anomaly Detection
